Main Application (main.py)

from fastapi import FastAPI, HTTPException from pydantic import BaseModel from typing import Optional, List from datetime import datetime # Initialize FastAPI app app = FastAPI( title="Recipe Book API", description="API for managing recipes" ) # ===== PYDANTIC MODELS ===== class RecipeBase(BaseModel): title: str description: str ingredients: List[str] instructions: str prep_time: int # minutes cook_time: int # minutes servings: int class RecipeCreate(RecipeBase): pass class Recipe(RecipeBase): id: int created_at: datetime class Config: from_attributes = True # ===== IN-MEMORY DATABASE ===== recipes_db = {} next_id = 1 # ===== ENDPOINTS ===== # CREATE: Add new recipe @app.post("/recipes/", response_model=Recipe) async def create_recipe(recipe: RecipeCreate): global next_id db_recipe = { "id": next_id, "created_at": datetime.now(), **recipe.dict() } recipes_db[next_id] = db_recipe next_id += 1 return db_recipe # READ: Get all recipes @app.get("/recipes/", response_model=List[Recipe]) async def get_all_recipes(): return list(recipes_db.values()) # READ: Get specific recipe @app.get("/recipes/{recipe_id}", response_model=Recipe) async def get_recipe(recipe_id: int): if recipe_id not in recipes_db: raise HTTPException( status_code=404, detail="Recipe not found" ) return recipes_db[recipe_id] # UPDATE: Modify recipe @app.put("/recipes/{recipe_id}", response_model=Recipe) async def update_recipe( recipe_id: int, recipe: RecipeCreate ): if recipe_id not in recipes_db: raise HTTPException( status_code=404, detail="Recipe not found" ) recipes_db[recipe_id].update( recipe.dict() ) return recipes_db[recipe_id] # DELETE: Remove recipe @app.delete("/recipes/{recipe_id}") async def delete_recipe(recipe_id: int): if recipe_id not in recipes_db: raise HTTPException( status_code=404, detail="Recipe not found" ) del recipes_db[recipe_id] return {"message": "Recipe deleted"}

Setup & Installation

1. Install Dependencies

pip install fastapi uvicorn pydantic

2. Run the Server

fastapi dev main.py
Note: Server runs at http://127.0.0.1:8000

3. Documentation

  • Swagger UI: http://127.0.0.1:8000/docs
  • ReDoc: http://127.0.0.1:8000/redoc

Key Components

  • Pydantic Models: Define recipe structure with validation
  • In-Memory DB: Simple dict for demo (use SQLAlchemy for production)
  • CRUD Endpoints: Create, Read, Update, Delete operations
  • Error Handling: HTTPException for not found recipes

API Usage Examples

POST /recipes/

Request Body:

{ "title": "Chocolate Chip Cookies", "description": "Classic cookies", "ingredients": [ "2 cups flour", "1 cup butter", "2 eggs", "1 cup chocolate chips" ], "instructions": "Mix, bake at 350¬∞F...", "prep_time": 15, "cook_time": 12, "servings": 24 }

Get All Recipes

GET /recipes/

Response:

[ { "id": 1, "title": "Pasta Carbonara", "description": "Italian classic", "ingredients": [...], "instructions": "...", "prep_time": 10, "cook_time": 20, "servings": 4, "created_at": "2024-11-28T..." }, { "id": 2, "title": "Caesar Salad", ... } ]

Get Single Recipe

GET /recipes/1

Response:

Update a Recipe

PUT /recipes/1

Request Body (entire recipe):

{ "title": "Pasta Carbonara (Updated)", "description": "Authentic Italian", "ingredients": ["spaghetti", "eggs", "bacon", "parmesan"], "instructions": "...", "prep_time": 10, "cook_time": 20, "servings": 4 }

Delete a Recipe

DELETE /recipes/1

Response:

{ "message": "Recipe deleted" }

Next Steps: Production Upgrade

The example uses in-memory storage. For production, add a database:

pip install sqlalchemy

Database Model Example

from sqlalchemy import create_engine from sqlalchemy.orm import Session from sqlalchemy.ext.declarative import declarative_base from sqlalchemy import Column, Integer, String, DateTime from datetime import datetime Base = declarative_base() class RecipeDB(Base): __tablename__ = "recipes" id = Column(Integer, primary_key=True, index=True) title = Column(String, index=True) description = Column(String) ingredients = Column(String) # JSON string instructions = Column(String) prep_time = Column(Integer) cook_time = Column(Integer) servings = Column(Integer) created_at = Column(DateTime, default=datetime.now) # Connect database engine = create_engine("sqlite:///recipes.db") Base.metadata.create_all(bind=engine) def get_db(): db = SessionLocal() try: yield db finally: db.close()
Pro Tips:
  • Use dependency injection: db: Session = Depends(get_db)
  • Add authentication with JWT tokens
  • Implement search/filter: @app.get("/recipes/search")
  • Use Docker for containerization
  • Add unit tests with pytest
